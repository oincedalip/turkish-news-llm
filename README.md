# turkish-news-llm
Fine tuning an LLM model using Turkish news articles dataset on Kaggle.

Dataset: [https://www.kaggle.com/datasets/dilemre/turkish-news-article](https://www.kaggle.com/datasets/dilemre/turkish-news-article)

Base Model: [https://huggingface.co/ytu-ce-cosmos/turkish-gpt2](https://huggingface.co/ytu-ce-cosmos/turkish-gpt2)
